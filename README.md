# vue-demos

vue 技术栈的 演示的一些 DEMO 超集。

## 项目结构

```
│  .gitignore
│  .postcssrc.js
│  babel.config.js
│  package-lock.json
│  package.json
│  README.md
│  tree.txt
│  vue.config.js
├─public
├─components // 子组件文件夹
└─src
    │  App.vue
    │  main.js
    │
    ├─assets
    │  └─css
    │
    ├─router
    │
    └─views
      └─ ScrollList.vue // 中奖连续滚动列表
      └─ KeyBoard.vue // 软键盘
      └─ HocParent.vue // 演示 $attrs、$listeners 传参
```
